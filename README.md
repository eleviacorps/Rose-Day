# Rose Day Web Page

A single-page interactive `Rose Day` experience built with plain HTML, CSS, and JavaScript.

## Features

- Animated rose bloom opening scene
- Multi-scene flow with smooth transitions
- Tap-to-reveal message interaction
- Envelope/letter interaction
- Built-in audio player cards
- Notes, feature cards, gallery, and ending quote screen
- Mobile responsive layout

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Run Locally

1. Clone the repository.
2. Open `rose-day.html` in your browser.

## Deploy

You can deploy this project as a static site using:

- GitHub Pages
- Netlify
- Vercel

Make sure `audio/` and `images/` folders are uploaded with `rose-day.html`.

## Notes

All content (messages, songs, image captions) can be edited inside `rose-day.html` in the `CONFIG` object.
